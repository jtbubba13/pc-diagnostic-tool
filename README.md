# 🛠️ PC Diagnostic & Anomaly Detection Tool

A modular Python-based diagnostics platform designed to detect system health issues, suspicious processes, and configuration risks across Windows environments.

## 🚀 Features
- 🔍 Process scanning (CPU, memory, execution path)
- ⚠️ Rule-based anomaly detection engine
- 🧠 AI-powered remediation recommendations (WIP)
- 💾 MySQL-backed scan history tracking
- 📊 Exportable reports (CSV)

## 🧱 Architecture
- Modular scanner framework
- Data processing via Pandas
- Persistent storage with MySQL
- Extensible rules engine

## 🛠️ Tech Stack
- Python
- Pandas
- psutil
- MySQL
- REST APIs (AI integration)
